# Expense Control Application

## Overview

This project is a simple expense control application developed using Python. It helps users manage their personal finances by allowing them to record, categorize, and review their expenses.

## Features

- **Add Expenses**: Allows users to record new expenses with a description and amount.
- **Categorize Expenses**: Users can assign categories such as "Entertainment" or "Food" to each expense for better organization.
- **Detailed Summary**: Provides a comprehensive summary of all recorded expenses, including totals and category breakdowns.

## Technologies Used

- **Python**: The primary programming language used for the application.
- **Pillow**: A Python Imaging Library used for handling images.
- **Tkinter**: The standard GUI library for Python, used to create the application's interface.
